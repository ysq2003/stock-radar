# 📈 Stock Radar

A-股历史行情可视化工具，输入股票代码，即时查看最近 6 个交易日的
最高价 / 最低价 / 收盘价走势图与数据明细表。

## 功能

- 最多同时查询 5 只股票
- 走势折线图 + 振幅填充，支持单独切换显示/隐藏各价格线
- 数据明细表格，涨跌幅、成交量一目了然
- 自动覆盖节假日，始终凑足 6 个交易日
- 查询历史记录，常用代码一键复用

## 数据来源

行情数据通过自建 Cloudflare Worker 代理极速数据（jisuapi.com）获取，
前端直接调用 Worker API，无需后端服务器。
